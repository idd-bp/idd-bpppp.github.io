Overview of the Program

Introduction

Interaction Design is a program suited for team workers, challenge lovers and creative thinkers. This program connects a digital world with the real one through revolutionary design solutions and creative and brave ideas. Interaction between a user and its product is one of the most vital parts of success and through this program, You will learn how to establish the most enjoyable and the most efficient interaction. This is a fast paced, competitive industry that is ready for You to conquer it.

What is learned

In this program, You will learn how to connect a user and the usable interface. You will learn how to avoid possible problems and solve existing ones. Interaction Design includes HTML,CSS and Java programming, graphic design, design theory, communication, information design, wireframing, 3D modelling and augmented reality studies. The program blends together all skills needed for a professional in the design industry, ready to overcome the hardest and most complicated problems with creativity, logic and team-work.

Resources

Interaction Design students use various software and hardware throughout the program, all of which is included with the program. Adobe Creative Cloud, Autodesk Maya, HTML,CSS, Java environments. The program also provides spaces and computers to work.

Courses

Information Architecture: IA420
Introduction to project building, client interviews and problem solving.
Interface Development: ID777
HTML, CSS, Javascript development.
3D Modelling: 3D111
Introduction to 3D modelling in Autodesk Maya.
Visual Design: VD222
Design theory, colour theory studies.
